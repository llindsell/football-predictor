# Context
This is a simple mobile optimized web app that tracks nfl football picks for a user against the spread each week. The user can also compare their picks against another user after they've submitted them. And they can see their rank on a weekly leaderboard and overall leaderboard.

# Key Features
- User can login with google
- User can submit their picks against the spread for a given week
- User can see their picks for a given week
- User can see the leaderboard for a given week
- User can see the overall leaderboard
- User can compare their picks against another user

# Tech Stack
